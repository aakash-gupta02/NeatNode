NeatNode includes several prebuilt TypeScript templates to help you kickstart development fast — whether you’re building a simple microservice, a full REST API, or a real-time server.

## Available Templates

---

> [!TIP]
> All templates share the same project structure — once you learn one, you’ll understand them all.

> [!NOTE]
> You can choose a template directly in the CLI prompt:
> ```bash
> npx neatnode
> ```
> and switch later by copying or merging configurations.
