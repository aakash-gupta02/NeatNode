# Welcome to NeatNode Docs

NeatNode is a CLI tool that scaffolds production-ready Node.js backends from opinionated templates.

Usage example:

```bash
npx neatnode
```

Official templates included with the project:

- Basic Express — minimal Express app with optional Todo CRUD
- REST API — full REST stack with auth, validation, services, and logging
- Socket.IO — Express + Socket.IO setup for real-time apps
