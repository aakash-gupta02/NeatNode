# Folder Structure (Guide)

This guide explains the folder layout used by templates and how to extend or modify it for custom projects.

Typical layout generated by a template:

```
<project-root>/
├─ package.json
├─ server.js
└─ src/
   ├─ app.js
   ├─ config/
   ├─ controllers/
   ├─ models/
   ├─ routes/
   ├─ services/
   ├─ middleware/
   └─ utils/
```
